## 🧩 2048 Game

A clean and interactive implementation of the classic **2048 puzzle game**, built with a focus on efficient game logic and a smooth user experience.

### 🚀 Features
- Dynamic tile movement and merging mechanics  
- Real-time score tracking  
- Responsive and intuitive controls  
- Smooth gameplay experience  

### 🛠️ Purpose
This project was developed to strengthen problem-solving skills and gain hands-on experience with:
- Grid-based algorithms  
- State management  
- UI design and user interaction  

---

A simple yet practical project that reflects core programming concepts through a well-known game.
